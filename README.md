# Installation:
1. Download [Fabric](https://maven.fabricmc.net/net/fabricmc/fabric-installer/1.0.0/fabric-installer-1.0.0.exe)
1. Install with MC version 1.20.4 and Loader 0.15.7 (with `Create profile` checked)
1. Launch the new profile, quit when it loads into the menu
1. Download this repository (`Code -> Download ZIP` or [click this link](https://github.com/Faupi/MinecraftMods/archive/refs/heads/main.zip))
1. Open the archive in WinRAR or Windows Explorer
1. Copy the folders `config`, `mods`, and `shaderpacks` into the profile's directory
    - By default it's `%APPDATA%/.minecraft`, you can run that with Win+R
1. Profit?
